Ask CompassDocs

Answers, not just links.

Ask a question in plain English. CompassDocs retrieves the most relevant documents and returns a synthesized answer—grounded only in your knowledge base, with inline citations you can click through to the source.

No API key? Search still works fully—it falls back to ranked keyword snippets instead of a written answer.

Access & governance

Who can read, write, and publish—your call.

CompassDocs is private by default. Four roles decide what each person can do, and an approval workflow keeps published knowledge trustworthy.

Strict · default

Review before it’s live

An editor’s change to a published doc becomes a pending change request. An approver reviews it, then publishes or rejects—so live knowledge is always vetted.

Open

Move fast

Prefer velocity? Admins can switch to open mode and let editors publish and update live docs directly, with no review step.

Get started

Up and running in about five minutes.

CompassDocs is self-hosted and open. Clone it, install, and run—it seeds sample content and creates your admin account on first launch.

Clone & install

Grab the repo and install dependencies. You’ll need Node.js 20+.

Run it

npm run dev starts the app on localhost—storage is a local SQLite file.

Sign in as admin

Use admin / admin, set a new password, then add your team and assign roles from the Admin console.

Turn on AI (optional)

Add an ANTHROPIC_API_KEY to enable synthesized answers. Search works without one.

bash — compassdocs 
$ git clone https://github.com/mattny20/CompassDocs.git
$ cd CompassDocs
$ npm install
$ npm run dev
  ▲ CompassDocs ready on http://localhost:3000
  → sign in with admin / admin
